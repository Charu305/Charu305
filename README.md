## 🚀 Featured Project

- [Used Car Price Prediction — HopeAI](https://github.com/Charu305/HopeAI-Car_price_prediction) — An end-to-end ML pipeline that predicts the resale price of used cars using supervised regression (Random Forest, XGBoost). Covers the full lifecycle: data collection, EDA, preprocessing, feature selection, model training (~92% R²), and deployment.

---

## 🔧 What I worked on in this project

- [Data Collection & EDA](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Data%20Analysis) — Explored 8,000+ real-world used car listings to uncover price drivers like car age and kilometres driven
- [Data Preprocessing](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Data%20Preprocessing) — Cleaned missing values, encoded categorical features, and engineered a `car_age` feature
- [Feature Selection](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Feature%20Selection) — Removed low-signal features using correlation analysis and importance scoring
- [Model Training & Evaluation](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Model%20Creation%20and%20Training) — Compared Linear Regression, Decision Tree, Random Forest, and XGBoost; best R² of ~0.92
- [Deployment](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Deployment%20Phase) — Serialised the model and built a live prediction interface using Streamlit / Flask
