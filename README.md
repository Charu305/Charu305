## 🔧 What I worked on — Used Car Price Prediction

- [Feature Selection](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Feature%20Selection) — Removed low-signal features using correlation analysis and importance scoring
- [Model Training & Evaluation](https://github.com/Charu305/HopeAI-Car_price_prediction/tree/main/Model%20Creation%20and%20Training) — Compared 4 regression models; best R² of ~0.92 with Random Forest
